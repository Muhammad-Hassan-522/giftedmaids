# Giftedmaids

Giftedmaids is a modern, responsive Angular application designed to manage bookings and user interactions efficiently. It integrates Google Calendar API for event management and EmailJS for handling contact and booking emails. The app is ready for deployment on any static hosting platform.

---

## Features

- Google Calendar integration for scheduling events
- Booking and contact email forms using EmailJS
- Responsive navigation with a Navbar and Footer
- Integration with Tidio live chat
- Ready for deployment on public hosting

---

## Tech Stack

- Angular 16+  
- TypeScript  
- Google Calendar API  
- EmailJS  
- Tidio live chat
