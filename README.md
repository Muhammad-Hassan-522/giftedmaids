## Project Overview

**GiftedMaids** is a web application that allows users to:

- Explore various house cleaning services
- Book cleaning sessions online
- Customize cleaning preferences and add optional services
- View service details, pricing, and schedules
- Contact the service team easily

This project is **user-friendly** and designed for **clients who need to book cleaning conveniently**.

---

## Features

### For Homeowners / Users

- **Browse Services:** See all cleaning options, including deep cleaning, move-in/move-out cleaning, and standard home cleaning.
- **Select Cleaning Type:** Choose the service that fits your needs.
- **Customize Booking:** Enter home details (rooms, bathrooms, kitchen, etc.) and select optional add-ons.
- **Schedule Cleaning:** Pick a date and time that works for you.
- **Booking Summary:** Review selected services and total cost.
- **Contact Support:** Call or email the team directly through the app.
- **FAQ Section:** Quickly find answers to common questions.

### For Admin / Service Providers

- Maintain service listings and pricing
- Track customer bookings and preferences
- View inquiries and manage customer communication

---

## How It Works

1. **Browse Services:** Users view the cleaning options on the website.
2. **Select Cleaning Type:** Choose from standard cleaning, deep cleaning, or move-in/move-out cleaning.
3. **Customize Booking:** Add home size details, frequency, and optional add-ons.
4. **Schedule Cleaning:** Pick a date and time.
5. **Confirm Booking:** Review all details and submit.
6. **Receive Confirmation:** Users get a confirmation of the appointment via the website or email.

## Developer Setup Instructions

Follow these steps to **run the GiftedMaids project locally**.

---

### Prerequisites

Before starting, ensure you have installed:

- **Node.js** (v16 or higher recommended)
- **Angular CLI** (v19.2.17 recommended)
- **Git**

---

### Steps to Run Locally

1. **Clone the repository**

   To get started, clone the repository to your local machine:

   ```bash
   git clone https://github.com/Muhammad-Hassan-522/giftedmaids.git
   cd giftedmaids
   ```

2. **Install project dependencies**

   Install all required Node.js packages:

   ```bash
   npm install
   ```

3. **Run the development server**

   Start the Angular development server:

   ```bash
   ng serve
   ```

   This will compile the project and start a local server.

4. **Access the app in your browser**

   Open your web browser and navigate to:

   http://localhost:4200

- You should see the GiftedMaids Home Cleaning Service App running locally.

- You can also view the live website at: [https://www.giftedmaidsct.com](https://www.giftedmaidsct.com)
